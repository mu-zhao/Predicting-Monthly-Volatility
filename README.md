# Predicting-Monthly-Volatility
1. Clean minute by minute stock price data using Numpy and Pandas package.
2. Use EWMA medel to predict monthly volatility, calibrate the model with gradient descent method.

Files:
1. stockdata2.csv: minute by minute stock prices (one year).
2. data_clean.py: built a class to clean data.
3. Monthly_volatility_report: calibrate model with gradeint descent method and showcase the data_clean package.
